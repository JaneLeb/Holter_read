# 🤖 Инструменты (стек технологий - обзор)
#### 🔄Техническая часть

Для полноценной работы стартапа дневник пациента должен быть доступен в интернете 24/7. В рамках студенческого акселератора пока не нужно покупать дорогие сервера или нанимать системного администратора. Все современные инструменты позволяют развернуть MVP дневника бесплатно за 5 минут.
Вот 3 лучших и самых простых способа поднять сервер для вашего Streamlit-дневника:


##### Вариант 1. Streamlit Community Cloud (Самый простой и бесплатный)

<img width="1392" height="1107" alt="image" src="https://github.com/user-attachments/assets/38168815-818f-4ae7-b565-839aa7d25dd9" />

https://docs.streamlit.io/deploy/streamlit-community-cloud/manage-your-account/sign-in-sign-out
Это официальный облачный хостинг от создателей Streamlit. Он создан специально для дата-сайентистов и стартапов, чтобы показывать свои MVP менторам и инвесторам.


* Цена: Абсолютно бесплатно.
* Как развернуть:
   1. Вы выкладываете код дневника на свой аккаунт в GitHub (в публичный репозиторий).
   2. Заходите на сайт share.streamlit.io.
   3. Авторизуетесь через свой GitHub, выбираете репозиторий и нажимаете кнопку Deploy.
   4. Через 2 минуты сервер готов. Вы получаете красивую постоянную ссылку вида https://streamlit.app.

##### ⚠️ Важный нюанс для стартапа (Как они свяжутся?)
Чтобы десктопное приложение врача на компьютере могло скачать то, что пациент ввел на сервере в интернете, им нужна общая база данных.
Поскольку на первом этапе не очень понятно как глубоко копать тему баз данных ( медицинские лданные должны быть зашифрованы, обезличены и использовать специальные сервера), на демо-дне можно применить хитрость («костыль» для MVP), которая работает безотказно и пишется в две строки кода:

   1. Сервер дневника в интернете записывает всё, что нажал пациент, не в локальный файл, а в бесплатную онлайн-таблицу (например, Google Sheets или Яндекс Таблицу). Для Streamlit есть готовый плагин st.connection("gsheets").

    
   3. Десктопное приложение врача при нажатии кнопки «Синхронизировать» просто скачивает эту Google-таблицу как обычный CSV-файл по прямой ссылке, находит там нужный ID пациента и выводит на экран.

Это решение разворачивается за один вечер, не требует настройки серверов баз данных, но при этом на защите выглядит как полностью рабочая интеграция через облако.
Что думаете насчет интеграции через Google / Яндекс Таблицы для демонстрации на демо-дне, или хотите сразу заложить архитектуру с полноценной базой данных?

Для создания красивого приложения с графиками на Python обычно используется комбинация из библиотеки для интерфейса (GUI) и библиотеки для построения самих графиков. Выбор зависит от того, нужен ли вам веб-интерфейс или классическое десктопное приложение. 

Лучшие современные связки и инструменты.Они позволяют создавать интерфейсы очень быстро:

Идеальный выбор для современных, стильных и интерактивных графиков, которые открываются в браузере.

##### 3. Сами библиотеки графиков
Отвечают за визуальную часть:

##### Современные графические фреймворки на Python обеспечивают стильный внешний вид:


##### 1. Веб-интерфейсы (Streamlit и Dash)


* Дизайн: Выглядят как дорогие современные панели мониторинга , веб-сервисы или SaaS-платформы (Software as a Service  — это облачные сервисы, где вы арендуете готовое программное обеспечение, оплачивая подписку)
* Фишки: Адаптивная верстка, поддержка мобильных устройств, идеальные шрифты, плавные CSS-анимации и встроенный переключатель темной/светлой темы.
  

##### 2. Десктопные интерфейсы (CustomTkinter и PyQt6)


* CustomTkinter: Разработан специально для решения проблемы устаревшего вида Tkinter. Элементы управления имеют закругленные углы, градиенты, современные плоские иконки и автоматическую поддержку темной темы Windows/macOS.
* PyQt6 / PySide6: Поддерживают кастомизацию через QSS (Qt Style Sheets) — это аналог веб-CSS. С его помощью интерфейс можно стилизовать под дизайн Windows 11, macOS или создать полностью уникальный футуристичный UI. На PyQt написаны такие современные программы, как VLC Media Player и Anki.
  

##### 3. Графики (Plotly и Seaborn)


* Plotly: Строит полностью интерактивные графики на базе JavaScript (D3.js). Они поддерживают плавное масштабирование, всплывающие подсказки при наведении и выглядят максимально современно.

Проект представляет собой не просто интерфейс, а высокотехнологичную интеграционную платформу. Врач в концепции не размечает данные вручную, а выступает в роли эксперта, который валидирует готовые результаты работы ИИ и C++ модулей.

Чтобы связать десктопный C++ движок фильтрации, инференс нейросети и электронную карту (МИС / ЕМИАС) в один современный интерфейс, скорее всего подойдет архитектура на базе FastAPI (бэкэнд) + Streamlit (фронтенд) или полностью автономное приложение.
